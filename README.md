# SP Autonomous Production Support

An enterprise-grade Agentic AI system designed to automate production support operations using LangGraph, Anthropic Claude, and the Model Context Protocol (MCP).

## Architecture Overview
- **Orchestrator**: LangGraph Multi-Agent Supervisor
- **Ingestion**: RabbitMQ (Event-driven)
- **Knowledge Base**: Vector Database + RAG (Confluence/Historical Tickets)
- **LLM**: Anthropic Claude 3.5 Sonnet
- **Integrations**: MCP Servers for Jira, Splunk, ServiceNow, Email, and Source Code
- **UI**: ReactJS Dashboard (Human-in-the-Loop)
